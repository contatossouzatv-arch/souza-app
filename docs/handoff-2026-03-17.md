## Problema corrigido no fim do dia

- Houve um crash no `DailyEvent` porque eu coloquei `isLowPerfMapDevice` antes do `useState` de `size`.
- Isso foi corrigido movendo o `useMemo` para depois do `useState` de `size`.
- Depois disso, `npm run build` voltou a passar.
